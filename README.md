# 📎 


## 서비스 소개
* 서비스명: 위치스 모바일 회의실 실시간 예약관리 앱
* 서비스설명: 인증된 직원들만 회의실 예약, 조회가 가능하며, 기존 예약자만 예약취소 할 수 있는 실시간 회의실 관리 앱
<br>

## 프로젝트 기간
2023.11.13 ~ 2023.11.24 (2주)
<br>

## 주요 기능
* 카카오 로그인 인증
* 회의실 에약 조회
* 회의실 예약 취소
* 회의실 예약 등록
<br>
